# Eleventy boilerplate with Tailwind and optional Alpine.js

## Basic setup
Install dependencies:
```
npm install
```

Run the code:
```
npm start
```

## Disable Alpine.js
To disable alpine.js open `src/_data/site.json` and search for 
```
"alpinejs" : true
```

Change `true` to `false`
```
"alpinejs" : false
```
